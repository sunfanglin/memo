# 喜马拉雅山脉边缘区域大气边界层结构特征及其对***的影响
## 思路
1. ABL 的南北差异，随地形的变化
2. 河谷内对流边界层的发展随着地形的变化
3. 对流 ABL 的发展对水汽和污染物传输的影响（限制）
4. 河谷南部低海拔地区对流 ABL 较低，水汽和污染物不易输送到高层
## 研究区域
1. Arun 河谷：珠峰站、曲当、陈塘；
2. 亚东、大峡谷
3. 西部河谷
4. 尼泊尔
